# whoami — Subclass Blurbs

One blurb per **Class / Subclass** combination. Class = the dial furthest from
neutral; subclass = the second-furthest (see `class-map.md`). 12 classes × 10
subclass options (the other five axes, each pole) = **120 combos**. Wildcard
never carries a subclass, so it has no entries here.

Use the matching blurb as the one-line identity on the character sheet and in
the profile summary. Class core traits are defined in `class-map.md`.

## Vanguard

- **Vanguard / Loremaster** — Runs ahead, but wants the full picture in hand: act decisively, and never thin out the detail in the report back.
- **Vanguard / Oracle** — Runs ahead and wants it brief: act decisively, report back in the fewest words that carry the result.
- **Vanguard / Pathfinder** — Acts fast, but lay the routes out first: move decisively once the options are on the table.
- **Vanguard / Lancer** — Pure forward motion: pick one line and commit, no deliberating, no fan of options.
- **Vanguard / Artificer** — Moves fast but wants the mechanism: act decisively, then show how it works.
- **Vanguard / Sorcerer** — Moves fast and trusts the result: act decisively, hand over the outcome, skip the derivation.
- **Vanguard / Bard** — A fast-moving partner who wants warmth: take the lead, but keep it human while you do.
- **Vanguard / Sentinel** — Fast and all business: take the lead, report the result, no rapport overhead.
- **Vanguard / Duelist** — Takes the lead and wants to be tested: act decisively, but push back hard when the direction is wrong.
- **Vanguard / Guardian** — Takes the lead with a wingman: act decisively, and back the call once it's made.

- **Marshal / Oracle** — Holds command and wants it brief: wait for the order, then give the distilled answer.

- **Marshal / Lancer** — Directs the work and wants one call: await the order, then commit to a single clear line.
