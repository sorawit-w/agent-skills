---
description: Build, view, or update your whoami collaboration profile
argument-hint: "[rerun]"
---

Run the `whoami` skill (`skills/whoami/SKILL.md`).

- **No argument** — if no profile exists, run the full interview. If a profile
  already exists, enter Show mode: present the summary + class and offer to
  correct something, re-run, or leave it.
- **`rerun`** — enter Re-run mode: show the existing profile, ask whether to
  confirm-and-adjust from it or start fresh, then run the interview.
