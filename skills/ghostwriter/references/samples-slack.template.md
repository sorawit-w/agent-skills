# Slack / Teams / DM voice samples — template

This is a **template with fake examples**. Your real samples never live in this folder — installed skill folders get overwritten on update, and real messages are personal data that must not end up in a repo.

**Setup:**

1. Copy this file to `~/.claude/ghostwriter/samples-slack.md`.
2. Replace the fake examples with 2–3 real Slack/Teams/DM messages **you wrote**. Casual-channel habits matter most here: capitalization, punctuation, emoji, fragments.
3. **Strip third-party PII** — other people's names and quoted messages get placeholders or deletion.
4. Don't clean them up. If you type lowercase without periods, the samples should show it — that's exactly what the skill needs to learn.

**Format:** one `## Sample` section per message, with a one-line context note.

---

## Sample 1

*Context: telling a teammate a build issue is fixed.*

> fixed, it was the cache key again 🙄 rerun and you should be good

## Sample 2

*Context: asking my manager for a quick decision.*

> hey, need a call on the retry limit before I merge — 3 like the old service or bump to 5? slight pref for 3

## Sample 3

*Context: declining a meeting in a group channel.*

> gonna skip this one, nothing on my side this week. ping me if anything for infra comes up
