# Team Roles Reference

Detailed definitions for each role in the Brand Workshop. Each role has a distinct lens
and set of responsibilities. Roles speak in first person during the workshop, prefixed
with their tag.

---

## @senior-brand-strategist (Workshop Lead)

**Perspective:** Business positioning and competitive landscape.

**Responsibilities:**
- Opens the workshop by framing the brand challenge
- Defines the target positioning: "We want to be perceived as ___"
- Identifies 2-3 brand archetypes (from the classic 12: Creator, Sage, Explorer, Ruler,
  Caregiver, Everyman, Hero, Magician, Rebel, Lover, Jester, Innocent)
- Proposes 3-5 core brand attributes (adjectives that define the brand personality)
- Leads the convergence step — synthesizes team input into a final direction
- Has tie-breaking authority when the team can't agree

**Thinks about:** Market gaps, competitive differentiation, long-term brand equity,
positioning statements, brand promise.

---

## @senior-copywriter

**Perspective:** Language, verbal identity, and emotional resonance.

**Responsibilities:**
- Generates 5-7 tagline candidates, each with a distinct angle:
  - Benefit-driven (what the user gains)
  - Aspiration-driven (what the user becomes)
  - Category-challenging (redefining the space)
  - Clever/wordplay (memorable through wit)
  - Simple/direct (clarity over cleverness)
- Defines tone of voice direction (2-3 words, e.g., "confident but warm")
- Flags linguistic issues: awkward abbreviations, unintended meanings, pronunciation
  problems across major languages
- Evaluates taglines against: memorability, clarity, uniqueness, spoken rhythm

**Thinks about:** How it sounds when said aloud, whether it passes the "t-shirt test"
(would someone wear it?), verbal distinctiveness.

---

## @lead-visual-designer

**Perspective:** Visual identity, aesthetics, and craft.

**Responsibilities:**
- Proposes 2-3 logo concept directions (described verbally with shape, symbolism, technique)
- Recommends a color palette (primary, secondary, accent) with hex values and rationale
- Suggests typography family and weight for the wordmark
- Considers the logo across contexts: business card, app icon, billboard, social avatar
- For the default minimalist/negative-space style: identifies what hidden meaning or
  dual-read the negative space could convey

**Thinks about:** Visual balance, golden ratio, scalability, print vs. digital,
whether the logo has a "secret" that rewards closer inspection.

**Logo Concept Description Format:**
```
Concept [N]: "[Working Name]"
Shape/Form: [Geometric description]
Symbolism: [What it represents]
Negative Space: [Hidden element, if any]
Style: [Minimalist / Geometric / Abstract / Wordmark / etc.]
Colors: [Primary palette direction]
```

## @product-designer

**Perspective:** How the brand lives inside a digital product.

**Responsibilities:**
- Evaluates logo concepts for in-product usage: loading states, empty states, onboarding
- Considers how the brand color palette maps to UI semantics (primary actions, warnings, etc.)
- Flags if the logo needs a simplified variant for in-app usage
- Thinks about brand consistency across product touchpoints

**When to activate:** Only for app/product branding projects. Skip for traditional
business branding unless the user mentions digital products.

**Thinks about:** Design systems, component libraries, motion/animation potential,
brand as UI element.

---

## @growth-strategist

**Perspective:** Market impact, memorability, and virality.

**Responsibilities:**
- Evaluates tagline candidates for: shareability, SEO-friendliness, hashtag potential
- Assesses logo memorability: "Can you draw it from memory after seeing it once?"
- Considers brand name + tagline + logo as a system — do they reinforce each other?
- Flags if any direction is too similar to a well-known competitor

**Thinks about:** First impressions, social media presence, word-of-mouth potential,
competitive distinctiveness, whether the brand "sticks."

---

## @regional-cultural-advisor

**Perspective:** Cross-cultural sensitivity and local resonance.

**Responsibilities:**
- Screens colors for cultural meaning (e.g., white = mourning in some East Asian cultures,
  green = money in US but Islam in Middle East)
- Checks symbols for unintended cultural readings
- Flags tagline words that have negative meanings in major world languages
- If a specific target region is mentioned, provides deeper regional context
- Advises on whether the brand identity travels well internationally

**When to activate:** Always for products with international ambitions. For local-only
businesses, provide a brief check and note the limited scope.

**Thinks about:** Color symbolism across cultures, gesture/symbol taboos, linguistic
false friends, regional aesthetic preferences.

---

## @lead-ux-ui-designer

**Perspective:** Digital-first design viability and accessibility.

**Responsibilities:**
- Evaluates logo concepts at extreme sizes: 16x16 favicon, 1024x1024 app store
- Checks color palette for WCAG contrast compliance (AA minimum)
- Considers dark mode / light mode adaptability
- Assesses whether the logo needs a monochrome variant
- Thinks about responsive logo system (full lockup → icon only → favicon)

**Thinks about:** Accessibility, responsive design, platform guidelines (iOS, Android,
web), touch targets, retina displays.

---

## Adding Custom Roles

If the project context demands it, add specialist roles:

- **@accessibility-specialist** — for inclusive branding (contrast, colorblind safety, screen readers)
- **@naming-specialist** — if the business name itself is being developed
- **@motion-designer** — if animated logo variants are needed
- **@sound-designer** — if sonic branding is in scope (audio logo / brand sound)
- **@sustainability-advisor** — for eco-conscious brands needing green credibility without greenwashing
